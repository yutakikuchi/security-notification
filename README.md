## security-notification

- Cameraを利用したリアルタイムで不審者を検知し、LINE Botに通知するサンプルアプリです
- Cloudの環境で不審者検知をする方法、Edge側で不審者を検知する方法のそれぞれを行います。

### 使用するもの

- Raspberrypi zero W
- Google Coral Edge TPU(USB Accelerator)
  - Edge側で不審者を検知するために必要
- ABEJA Platformアカウント
- LINE Developersアカウント
